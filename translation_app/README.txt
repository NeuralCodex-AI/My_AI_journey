Translation App (Streamlit + GoogleTranslator + gTTS)

This is a simple translation web application built using Streamlit, deep-translator, and gTTS (Google Text-to-Speech). It allows users to translate text between multiple languages and also listen to the translated text as audio.

Features:

Translate text between multiple languages
Convert translated text into speech (audio)
Simple user-friendly interface using Streamlit
Fast translation using GoogleTranslator
Audio generation using gTTS

Technologies Used:

Python
Streamlit
deep-translator (GoogleTranslator)
gTTS
