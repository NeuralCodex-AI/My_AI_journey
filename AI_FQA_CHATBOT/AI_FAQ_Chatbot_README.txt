AI FAQ Chatbot Using NLP and Cosine Similarity

Project Overview The AI FAQ Chatbot is a Natural Language Processing
(NLP) based chatbot developed in Python. It answers frequently asked
questions related to Artificial Intelligence using TF-IDF Vectorization
and Cosine Similarity.

Objectives - Understand NLP fundamentals - Perform text preprocessing
using NLTK - Implement TF-IDF Vectorization - Apply Cosine Similarity -
Build a simple FAQ chatbot

Technologies Used - Python - NLTK - Scikit-Learn

Working Procedure 1. Create FAQ dataset 2. Preprocess text 3. Apply
TF-IDF Vectorization 4. Accept user input 5. Calculate similarity 6.
Find best match 7. Display answer

Advantages - Easy to understand - Fast response time - Beginner
friendly - Demonstrates NLP concepts

Limitations - Uses predefined FAQs - Limited knowledge base - Cannot
generate new answers

Future Enhancements - Streamlit web app - Voice support - Multi-language
support - Database integration

Conclusion This project demonstrates how NLP techniques such as
tokenization, stopword removal, TF-IDF, and cosine similarity can be
used to build an intelligent FAQ chatbot.
