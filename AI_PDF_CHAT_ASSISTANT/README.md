# AI PDF Chat Assistant

An AI-powered PDF Chat Assistant built using **Python, Streamlit, LangChain, Gemini API, and FAISS**. This application allows users to upload one or multiple PDF documents, ask questions in natural language, generate summaries, quizzes, and MCQs, download summaries, and maintain chat history using a Retrieval-Augmented Generation (RAG) pipeline.

## Features

* Upload multiple PDF files
* Chat with PDF content
* Generate document summaries
* Generate quizzes and MCQs
* Download summary
* Maintain chat history
* Built using RAG with FAISS and Gemini API

## Technologies Used

* Python
* Streamlit
* LangChain
* Gemini API
* FAISS
* PyPDFLoader
* Google Generative AI Embeddings

## Installation

```bash
pip install -r requirements.txt
streamlit run pdf.py
```
